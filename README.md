# Recruitment System

Проект создан для автоматизации процессов управления кадрами в компании, где несколько HR-специалистов работают над закрытием вакансий для 10 различных отделов. Система предназначена для облегчения поиска кандидатов и оптимизации HR-работы, а также для повышения мотивации HR-специалистов посредством учета KPI.

## Архитектура системы

Система состоит из двух частей: фронтенд и бэкенд. В данном проекте реализована только серверная часть (бэкенд) с использованием языка программирования C#.

### Сущности системы

Кандидат: Класс, представляющий кандидата с параметрами, такими как имя, должность и статус.
Вакансия: Класс, представляющий вакансию с информацией о ее статусе и связанных кандидатах.
HR-специалист: Класс для представления HR-специалиста с KPI на основе закрытых вакансий.
Отдел: Класс для представления отделов компании с привязанными к ним HR-специалистами и вакансиями.
Вакансия: Класс дял представления вакансий компании с привязанными к ним отделами компании и кандидатами на вакансию.
